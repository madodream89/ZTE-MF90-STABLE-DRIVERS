Скрипт не является программой смены IMEI, а является автоматизацией процесса смены IMEI.

Драйверы:
Настоятельно рекомендую сразу отключить подпись драйверов скриптом «driver_sig_OFF_win_7_8_10» и перезагрузить компьютер!
Ставим драйвера Drivers_install.
Дополнительные драйверы для ZTE лежат C:\MF90PLUS\soft\MF90Drivers и C:\MF90PLUS\soft\ADBdriver. Они могут понадобится в процессе работы скрипта.
PS: После всех манипуляций с роутером, рекомендую вернуть компьютер в нормальный режим работы - запустить правами администратора «driver_sig_ON_win_7_8_10»

Скрипты: 
_ZTEMF90+.exe - смена IMEI / FIX TTL / FTP

Как работает (TTL, FTP): 
Запускам «ZTEMF90+.exe». Жмем ADB ON. Теперь ждем окошко «success».
TTL : Жмем необходимый фикс 64/128.
FTP : Для автозапуска FTP жмем «FTP A-START»
Настройки применяться только после презегрузки роутера.

Генерация IMEI NOKIA LTE
Идем http://www.nokiaport.de/tacdatabase/index.php?s=imeitools
В «Generate IMEI» выбираем «640 Lumia (LTE)» или нажимаем Generate.

Как работает (смена IMEI): 
Установка драйвера: запускаем Install_Driver.
Запускаем _ZTEMF90+.exe.  Водим корректный новый IMEI (15 цифр без букв, пробелов или чего либо еще. только цифры!) и нажмите Enter. 
После завершения работы скрипта роутер будет автоматически перезагружен. Все. (вся работа скрипта занимает примерно 4 минуты). Приложение QPST теперь можно удалить. 
PS Бэкапы хранятся в папке backup.
(!) Возможные проблемы обычно на Windows x64
Возможно у Вас не установлен корректный драйвер. Откройте диспетчер устройств
Если есть устройство ZTE LTE Device Mass Storage Filter с ошибкой, то на «ZTE LTE Device Mass Storage Filter» -> правой кнопкой -> свойства -> вкладка «драйвер» -> кнопка «удалить» -> ставим галку «удалить программы драйверов для этого устройства». 
Если есть неопознанное устройство QHSUSB_DLOAD, то обновите его драйвер вручную: Правой кнопкой -> Обновить драйвер ->  Поиск и установка драйвера вручную ->  Выбрать из списка уже установленных  драйверов ->  Показать все устройства -> Далее -> Установить с диска -> Выбрать папку soft\MF90Drivers -> ZTE Mobile Broadband Diagnostics Port...

Диспетчер устройств.
Откройте диспетчер устройств компьютера (ищите в панели управления компьютером), разверните раздел «Порты (COM и LPT)» и найдите в нем «ZTE Mobile Broadband Diagnostics Port (COMXX)», где XX - это номер порта. Например «(COM17)» означает, что порт Вашего роутер 17. Отнеситесь к этому пункту внимательно. Не закрывайте это окно. На каждый запрос скрипта «Enter ZTE Diagnostics Interface» необходимо смотреть его в диспетчере устройств (т.к. он может меняться), вводить эти цифры и нажимать Enter. Если запрос уже появился, а в диспетчере устройств его еще нет, то нужно подождать (не вводите ничего)! Везде нужно вводить только цифры порта (без COM)!
